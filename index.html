ClientXServer

Nessa aula entendemos o que é o lado do cliente e o lado do servidor, vimos o que é redenrizado para o cliente e como o navegador
busca essas infomações no servidor, entendemos como usar o HTML para criar a interação do usúario com o servidor.
Abaixo temos um trecho de um artigo postado sobre Client e Server e como funciona, você também pode clicar no link e ir para o artigo para ler mais sobre:

"O modelo Cliente-Servidor é uma arquitetura de rede amplamente utilizada na computação moderna. Nesse modelo,
os dispositivos de rede são divididos em duas categorias principais: o cliente e o servidor.
O cliente é o dispositivo que solicita serviços ou recursos do servidor, enquanto o servidor é o dispositivo que fornece esses serviços ou recursos.
Neste artigo, vamos explorar em detalhes o que é o modelo Cliente-Servidor, como ele funciona e quais são suas principais características."
